template<class T>
class Dtk::Widget::DAnchors< T >

DAnchors 是一个模板类，在 DAnchorsBase 的基础上保存了一个控件指针， 将控件和锚定绑定在一起使用，相当于把“锚线”属性附加到了控件本身

重载了 “->”、“*”、“&” 等运算符，用于把 DAnchors 这层封装透明化，尽量减少使用 DAnchors<QWidget> 和直接使用 QWidget* 对象的区别。
