详细描述

DRecentManager 是用来管理最近文件列表的类，提供了添加与删除文件项。

遵循 freedesktop 标准，在本地 share 目录存放，文件名为: recently-used.xbel，所以每个用户都有不同的列表。

该类的存在就是为 deepin 应用提供一个工具类，方便让打开的文件添加到最近文件列表中。
