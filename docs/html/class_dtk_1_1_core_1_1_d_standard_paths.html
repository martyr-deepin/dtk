◆ Mode

enum Dtk::Core::DStandardPaths::Mode

DStandardPaths支持的路径产生模式。

枚举值
Auto 

和Qt标准的行为表现一致。

Snap 

读取SNAP相关的环境变量，支持将配置存储在SNAP对应目录。

Test 

和Qt标准的行为表现一致，但是会开启测试模式，参考QStandardPaths::setTestModeEnabled。
