对于每种控件，均由一个id和对应的构造函数来组成，DSettingsWidgetFactory内置了许多基础的控件，包括：
checkbox/lineedit/combobox/spinbutton/buttongroup/radiogroup/slider等。 更多...

void Dtk::Widget::DSettingsWidgetFactory::registerWidget ( const QString &  viewType,
std::function< WidgetCreateHandler >  handler

DSettingsWidgetFactory::registerWidget 向Factory注册一种类型viewType。注册该类型后，可以通过在DSettings的DOption中，设置type为对应的viewType，这样Factory在构建viewType对应的控件时，会调用handler所指定的方法来构建。

参数
viewType控件类型，建议为全局唯一的字符串，且不要和内部类型重复，参考 Dtk::Widget::DSettingsWidgetFactory
handler自定义的控件构建函数， 参考 Dtk::Widget::DSettingsWidgetFactory::WidgetCreateHandler
