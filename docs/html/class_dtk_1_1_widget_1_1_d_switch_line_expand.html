详细描述

DSwitchLineExpand 类提供了一种通过开关按钮来展开/合上显示内容的控件。 控件由标题栏和内容两部分组成，标题栏左侧显示控件标题，右侧显示一个开关控件， 用户通过点击右侧的开关控件，来控制标题栏下面的内容部分展开还是合上。
