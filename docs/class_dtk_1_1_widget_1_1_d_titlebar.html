详细描述

Dtitlebar是Dtk程序通用的标题栏组件，用于实现标题栏的高度定制化。

参数
父组件，一般为标题栏所在的窗口一般情况下，请使用Dtk::Widget::DMainWindow::titlebar()来获取已经自动初始化的标题栏， 不要自己来创建这个标题栏。

在文件 dtitlebar.h 第 32 行定义.

返回
自定义控件

◆ menu()

QMenu * Dtk::Widget::DTitlebar::menu ( ) const

获取和标题栏关联的应用查询菜单。
