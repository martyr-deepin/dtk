Deepin Tool Kit (Dtk) is the base devlopment tool of all C++/Qt Developer work on Deepin.

Dependencies

Build dependencies

  • Qt >= 5.6

Installation

Build from source code

  1. Make sure you have installed all dependencies.
  2. Build:
$ mkdir build
$ cd build
$ qmake ..
$ make
  1. Install:
$ sudo make install
